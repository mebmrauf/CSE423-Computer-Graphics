# Eight Way Symmetry
<p>dx = x<sub>2</sub>-x<sub>1</sub></p>
<p>dy = y<sub>2</sub>-y<sub>1</sub></p>

### Quadrants
```
Q1 --> dy > 0, dx > 0
Q2 --> dy > 0, dx < 0
Q3 --> dy < 0, dx < 0
Q4 --> dy < 0, dx > 0
```

### Zone
```
|dx| > |dy| --> near to X axis
|dy| > |dx| --> near to Y axis
```

#### Example
```
Q. dx = 3, dy = 4

dx > 0, dy > 0
Selected quadrant Q1

|dy| > |dx| --> near to y axis
Selected zone 1

```
